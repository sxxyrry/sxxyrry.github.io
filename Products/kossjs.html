⚡ 快速开始 · Python 集成示例

只需三步： 下载平台动态库 → 获取 kossjs_interface.py → 编写脚本执行 JS。 
from kossjs_interface import KossJS

# 创建 KossJS 实例
koss = KossJS()

# 执行 JavaScript 表达式
result = koss.eval("(a, b) => a * b")(12, 7)
print(result)      # 输出: 84

# 设置全局变量，跨语言交互
koss.set_global("appName", "KossJS Demo")
js_code = """
    const greet = `Hello from ${appName}`;
    greet.toUpperCase();
"""
print(koss.eval(js_code))  # HELLO FROM KOSSJS DEMO

# 使用内置 Fetch API (异步)
koss.eval('''
fetch("https://api.github.com/repos/KossJS/KossJS")
  .then(res => res.json())
  .then(data => console.log("⭐ stars:", data.stargazers_count))
  .catch(e => console.error(e));
''')
import time; time.sleep(1.5)   # 等待异步完成

# 运行 .js 文件
# result = koss.run_file("./script.js")

koss.destroy()   # 显式销毁实例
高阶能力：注册 Python 函数给 JS 调用，支持文件模块加载，完全兼容 CommonJS 与 ES Module。

技术架构

架构层次
应用程序 (Python/C++/C#/Java) → KossJS 接口封装 → C ABI 稳定接口 → KossJS 核心引擎 (Rust) → Boa JS 引擎 + Tokio 异步运行时 + 网络栈

核心组件

实例管理器、JS 编译器、值转换器、模块加载器、Fetch 适配器、异步事件循环桥接。

核心依赖

Boa (ECMAScript 引擎), Tokio, rustls, serde, 以及 Node.js 模块 Shim 层。

构建工具链

Cargo, cbindgen (自动生成 C 头文件), GitHub Actions（跨平台编译）
