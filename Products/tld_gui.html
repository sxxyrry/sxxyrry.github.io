为普通用户设计

🎯 产品定位
TTHSD GUI 是基于 TTHSD 核心开发的图形界面版本，专为不需要编程知识的普通用户设计。使用 Python + PyWebview + Flask + Web 技术实现，提供直观易用的下载体验。
🖱️

图形化操作

无需命令行，通过直观的图形界面完成所有下载操作，适合所有用户

📋

批量下载

支持批量添加下载任务，自动管理下载队列，一键下载多个文件

📊

实时进度

可视化下载进度显示，实时速度、剩余时间一目了然

⏸️

暂停/恢复

支持暂停和恢复下载，断点续传不怕网络中断

💡 架构说明
GUI 版本使用 PyWebview 创建原生窗口，Flask 提供本地 Web 服务，前端使用现代 Web 技术构建 UI，后端调用 TTHSD 核心动态库进行下载。

主要功能

🔗 多协议支持

支持 HTTP/HTTPS、FTP、BitTorrent、Magnet、ED2K 等多种下载协议

⚡ 多线程下载

自动分片多线程下载，充分利用带宽，提升下载速度

📁 文件管理

自定义保存路径，自动创建目录，智能文件命名

🔔 任务通知

下载完成通知，错误提醒，任务状态实时更新

⚙️ 参数配置

可配置线程数、分块大小、User-Agent 等下载参数

📜 下载历史

记录下载历史，方便查看和管理已下载文件

TTHSD GUI vs TTHSD Core

TTHSD GUI

✅ 图形界面，无需编程
✅ 开箱即用
✅ 适合普通用户
⚠️ 功能相对固定

TTHSD Core / Next

✅ 核心下载引擎
✅ 高度可定制
✅ 多语言绑定
⚠️ 需要编程集成

💡 选择建议
如果您是普通用户，只需要下载功能，推荐使用 TTHSD GUI。
如果您是开发者，需要将下载功能集成到自己的应用中，推荐使用 TTHSD Core / Next。
